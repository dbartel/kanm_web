# Programming Director page

Programming Director needs to have the following abilities:

1. Audit Shows (PSAs/StationIds etc) & give strikes 
2. Audit Djs (Points) & give strikes

## Implementation: Tab view

### Show Tab
- Searchable list of all shows with playlists
- List of shows whose last playlist is "bad" in some way
- Ability to strike a show
- Ability to remove a show

### DJ Tab
- Searchable list of all djs with point amount
- Indicate which DJs do not have enough points/have too many strikes
- Ability to give/clear points
- Ability to strike a dj
