KANM 
============

## Frontend

- AngularJS
- AngularUI Bootstrap
- AngularUI Router

## Backend

- PHPSlim
- MySQL

## Configuration

Define a **config.ini** file in src/api with the following layout

		[database]
		dsn = "" ; datastore name e.g. mysql:host=localhost;dbname=db
		user = "" ; database user name
		password = "" ; database password
